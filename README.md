#
For starting website locally, from inside "hack-a-blog" , open 2 terminals

In terminal 1, 
1. cd frontend
2. npm i
3. npm start

In terminal 2, 
1. cd backend
2. npm i
3. npm run dev

Shortcut

cd frontend && npm i && npm start

cd backend && npm i && npm run dev
